React Picsum Gallery

A simple and responsive image gallery application built with React that fetches images from the Picsum Photos API and displays them in a grid layout.

Features
Fetch images from API using Axios
Uses React hooks (useState, useEffect)
Responsive image grid layout
Displays image with author name
Click image to open in a new tab
Simple and clean UI
Tech Stack
React JS
Axios
CSS (Flexbox)
Vite
API Used

Picsum Photos API
https://picsum.photos/v2/list
